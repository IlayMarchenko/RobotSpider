# Electronics

## List of components
1. Servo motors Tower Pro MG995 x12
2. Batteries Li-ion 18650 x2
3. DC-DC converter 12A from 5-40V to 1.2-36V
4. LDO from 4.5-7V to 3.3
5. CC1352R1 Texas Instruments
6. PCA9685 (servo motors driver)
